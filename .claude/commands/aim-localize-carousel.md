# 🌍 Копировать / локализовать / адаптировать карусель (aim_localize_carousel)

Используй MCP инструмент `aim_localize_carousel`.

Аргументы: $ARGUMENTS

Обязательные параметры:
- `url` — ссылка на оригинальную карусель
- `mode` — режим:
  - `copy` — рерайт (те же триггеры, другой текст)
  - `localize` — перевод на русский с адаптацией под РФ/СНГ
  - `adapt` — адаптация под новую тему/нишу

Опционально:
- `targetTopic` — новая тема (только для `adapt`)
- `slideCount` — количество слайдов
- `designTheme` — тема дизайна 1-8 для авторендера PNG
- `outputDir` — папка для PNG

Если режим не указан — уточни у пользователя.
