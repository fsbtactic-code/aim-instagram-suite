# ⏱️ Детектор провисаний монтажа (aim_extract_pacing)

Используй MCP инструмент `aim_extract_pacing`.

Аргументы: $ARGUMENTS

Нужен один из параметров:
- `videoPath` — абсолютный путь к локальному видеофайлу
- `url` — ссылка на видео

Опционально: `slowThresholdSec` — порог "провисания" в секундах (по умолчанию 4 сек).

Выведи:
- Таймкоды всех смен кадра
- Провисания (где зритель засыпает) с рекомендациями по ускорению
- Общую оценку динамики монтажа
