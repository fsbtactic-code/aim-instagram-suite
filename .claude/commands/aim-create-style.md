# aim-create-style

Запускает интерактивный мастер создания кастомного стиля для каруселей.

**Использование:** Вызови `aim_create_style` без параметров — начнётся пошаговый опрос:

- Шаг 1: Настроение / вайб (6 вариантов)
- Шаг 2: Цветовая схема (тёмный/светлый + HEX цвета)
- Шаг 3: Стиль шрифта (5 вариантов)
- Шаг 4: Плотность контента и эмодзи
- Финал: Готовый CSS-стиль + JSON-конфиг, сохранённый в ~/.aim_styles/

Результат применяется в `aim_render_premium_carousel` через параметр `brandColorOverlay`.

## Запуск

```
Запусти aim_create_style — создадим мой фирменный стиль карусели
```

## Продвинутые варианты

```
# Быстрый старт с готовым набором:
Создай style Luxury Gold: mood=luxury, colorBase=dark, primaryHex=#C9A84C, fontStyle=serif, contentDensity=minimal

# Полный опрос:
Запусти мастер стилей aim_create_style с шага 1
```
