# ✍️ Генерация сценария по структуре успешного видео (aim_generate_script)

Используй MCP инструмент `aim_generate_script`.

Аргументы: $ARGUMENTS

Нужны два параметра:
- `referenceMdPath` — путь к .md файлу, созданному командой `/project:aim-analyze-viral`
- `targetTopic` — тема/ниша для нового сценария (например: "фитнес для занятых мам")

Если не указано — уточни у пользователя.

Выведи готовый сценарий с разбивкой по секундам и указанием использованных триггеров.
