# 📊 Индекс Виральности ВИДЕО 0-100 (aim_score_virality)

Используй MCP инструмент `aim_score_virality`.

Аргументы: $ARGUMENTS

Нужен один из параметров:
- `videoPath` — абсолютный путь к локальному видеофайлу
- `url` — ссылка на видео

Опционально: `context` — ниша и целевая аудитория (повышает точность).

Критерии оценки:
🪝 Хук (25%) · ⚡ Динамика (20%) · 🎵 Звук (15%) · 💡 Ценность (15%)
❤️ Эмоции (12%) · 🎨 Визуал (8%) · 📢 CTA (5%)

Выведи балл, разбивку по критериям и Топ-3 правки с прогнозом прироста индекса.
