# 🪝 Анализ и усиление хука видео (aim_analyze_hook)

Используй MCP инструмент `aim_analyze_hook`.

Аргументы: $ARGUMENTS

Нужен один из параметров:
- `videoPath` — абсолютный путь к локальному видеофайлу
- `url` — ссылка на видео

Анализ сфокусирован только на первых 5 секундах.

Выдай 5 конкретных усиленных вариантов хука с объяснением какой психологический триггер используется в каждом.
