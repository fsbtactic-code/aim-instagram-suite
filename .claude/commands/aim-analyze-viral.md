# 🕵️ Реверс-инжиниринг вирального видео конкурентов (aim_analyze_viral_reels)

Используй MCP инструмент `aim_analyze_viral_reels`.

Аргументы: $ARGUMENTS

Нужны два параметра:
- `url` — ссылка на видео (Instagram Reel, TikTok, YouTube Shorts, VK)
- `outputMdPath` — путь для сохранения .md отчёта (например: `C:\Users\Alina\Desktop\viral.md`)

Если что-то не указано — уточни у пользователя.

После анализа выведи резюме: структура, главный триггер, формула успеха.
