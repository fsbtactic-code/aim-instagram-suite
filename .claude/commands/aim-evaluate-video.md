# 🎬 Оценка виральности видео (aim_evaluate_video)

Используй MCP инструмент `aim_evaluate_video` для анализа видео.

Аргументы: $ARGUMENTS

Если пользователь указал путь к видео — передай его в `videoPath`.
Если не указал — попроси абсолютный путь к видеофайлу (mp4, mov, avi).

После анализа выведи структурированный отчёт:
- 🪝 Сила хука (первые 3 секунды)
- ⏱️ Удержание внимания по хронометражу  
- ❤️ Эмоциональный резонанс
- 📊 Индекс виральности (0-100)
- ✅ Топ-3 рекомендации по улучшению
