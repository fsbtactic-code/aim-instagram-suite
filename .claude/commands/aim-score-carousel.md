# 📊 Индекс Виральности КАРУСЕЛИ 0-100 (aim_score_carousel_virality)

Используй MCP инструмент `aim_score_carousel_virality`.

Аргументы: $ARGUMENTS

Нужен один из параметров:
- `url` — ссылка на карусель Instagram
- `slidesDir` — путь к папке со слайдами PNG/JPG

Опционально:
- `context` — ниша и целевая аудитория
- `goal` — цель: `sales` / `subscribers` / `reach` / `engagement` / `trust`

Критерии (специфичные для carousel-формата):
🎯 Первый слайд/Хук (25%) · 🔀 Воронка/Структура (22%) · 🎨 Дизайн (18%)
💡 Ценность (17%) · ❤️ Триггеры (10%) · 📢 CTA-слайд (8%)

Выведи балл, разбивку, save-rate прогноз и рекомендации.
